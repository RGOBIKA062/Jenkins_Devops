## 🎓 Integration Instructions (5 Minutes)

### 1️⃣ Import Component
```javascript
// In client/src/pages/FacultyDashboard.jsx
import AIFacultyDashboard from '../components/AIFacultyDashboard';
```

### 2️⃣ Add Tab Button
```jsx
<button onClick={() => setActiveTab('ai-assistant')}>
  🧠 AI Assistant
</button>
```

### 3️⃣ Add Tab Content
```jsx
{activeTab === 'ai-assistant' && (
  <AIFacultyDashboard facultyId={user._id} />
)}
```

### 4️⃣ Test It!
```bash
# Terminal 1: Start backend
cd server && npm start

# Terminal 2: Start frontend
cd client && npm run dev

# Open http://localhost:8080
# Login → Click "AI Assistant" tab → Enjoy! 🎉
```
